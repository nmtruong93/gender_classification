### How to run
1. Create folder named `data` in the root directory
   `data` folder directory should be like this
   ```
    data
    ├──trainingData.csv
    ├──trainingLabels.csv
   
2. Install python 3.10
3. Install the required packages
    ```bash
    pip install -r requirements.txt
    ```
4. Run the main.py file
    ```bash
    python main.py
    ```
